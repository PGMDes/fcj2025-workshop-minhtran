### Kết quả đạt được tuần 1:

- Hiểu AWS là gì và các khái niệm và dịch vụ mà AWS cung cấp:

  - Điện toán đám mây.
  - Sự khác biệt của AWS
  - Cách để bắt đầu một hành trình lên mây
  - Hạ tầng toàn cầu của AWS
  - Công cụ quản lí AWS Services
  - Cách tối ưu hóa chi phí trên AWS

- Đã tạo và cấu hình AWS Free Tier account thành công.

- Làm quen với AWS Management Console và biết cách tìm, truy cập, sử dụng dịch vụ từ giao diện web.

- Cài đặt và cấu hình AWS CLI trên máy tính bao gồm:

  - Access Key
  - Secret Key
  - Region mặc định
  - MFA cho tài khoản
  - Admin Group
  - Admin User
  - AWS Support

- Biết cách thiết lập các Budget:
  - Cost Budget
  - Usage Budget
  - RI Budget
  - Saving Plan Budget
  - Clean tài nguyên

- Hiểu rõ về các gói hỗ trợ và cách truy cập AWS Support
  - Gói Basic
  - Gói Developer
  - Gói Business
  - Gói Enterprise
